Definicion de conceptos de CSS

Enero 13, 2020

CSS ayuda a los desarrolladores web a crear una apariencia uniforme en varias páginas de un sitio web. Además, CSS facilita el cambio de estilos en varias páginas a la vez. Por ejemplo, un desarrollador web puede querer aumentar el tamaño de texto predeterminado de 10pt a 12pt para cincuenta páginas de un sitio web.

Codigos de CSS

Enero 17, 2020

La propiedad de fondo en CSS le permite controlar el fondo de cualquier elemento (lo que pinta debajo del contenido de ese elemento). Es una propiedad abreviada, lo que significa que le permite escribir lo que serían múltiples propiedades CSS en una.
